
# Cm Pentatonic Scale for B-flat Harmonica

## Ascending:
```
-4    -5      6      -6      7     -8
 C     Eb     F      G      Bb     C
```

## Descending:
```
-8     7     -6      6     -5    -4      4     -3''    -3
 C     Bb     G      F     Eb     C      Bb     G      F
```

## Key Notes:
- **-3'' (Draw Bend): G** is achieved by bending the draw on hole 3.
- **-3 (Draw): F** is played normally on hole 3.
